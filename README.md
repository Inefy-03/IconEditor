<h1 align="center">IconEditor</h1>

<p align="center">
  一个 MTZ 和 Module 图标包编辑器
</p>

IconEditor 采用 [Miuix](https://github.com/compose-miuix-ui/miuix) 设计，用于创建、导入、编辑和导出图标包项目，支持 MIUI/HyperOS `.mtz` 主题和 Module `.zip` 图标包

## 主要功能

- 导入和编辑 MTZ 主题、Module 图标包
- 管理图标主题/模块信息
- 添加、替换和切换图标
- 导出修改后的主题文件
- 支持简体中文和英文

### 支持的格式

| 格式 | 导入 | 导出 |
| --- | --- | --- |
| MTZ 主题（`.mtz`） | 支持 | 支持 |
| Module 图标包（`.zip`） | 支持 | 支持 |

Module 压缩包必须包含顶层 `icons` 文件。

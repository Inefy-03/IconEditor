<h1 align="center">IconEditor</h1>

<p align="center">
  An app for organizing and editing Android icon themes.
</p>

IconEditor helps you manage icon theme projects, import and edit MTZ themes and Module icon packages, adjust icons, and export the updated theme files.

## Features

- Manage icon theme projects
- Import and edit MTZ themes and Module icon packages
- Add, replace, and organize icons
- Export updated theme files
- Match icons with installed applications
- Support Simplified Chinese and English

IconEditor preserves as much unmodified content from the original theme as possible.

The interface uses [Miuix](https://github.com/compose-miuix-ui/miuix).

## Implementation

- Kotlin, Jetpack Compose, and Miuix `0.9.3`
- Miuix `MiuixTheme`, `Scaffold`, and Navigation3 for theme, overlay, and page navigation integration
- Miuix overscroll for page scrolling and root-page boundary gestures

Implementation follows the [Miuix documentation](https://compose-miuix-ui.github.io/miuix/)
and the [v0.9.3 release notes](https://github.com/compose-miuix-ui/miuix/releases/tag/v0.9.3).

## Project status

IconEditor is still under active development. Feedback and suggestions are welcome.

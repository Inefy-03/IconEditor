<h1 align="center">IconEditor</h1>

<p align="center">
  An MTZ and Module icon package editor
</p>

IconEditor uses [Miuix](https://github.com/compose-miuix-ui/miuix) to create, import, edit, and export icon-pack projects for MIUI/HyperOS `.mtz` themes and Module `.zip` packages.

## Features

- Import and edit MTZ themes and Module icon packages
- Manage theme and module information
- Add, replace, and switch icons
- Export updated theme files
- Support Simplified Chinese and English

### Supported formats

| Format | Import | Export |
| --- | --- | --- |
| MTZ theme (`.mtz`) | Yes | Yes |
| Module icon package (`.zip`) | Yes | Yes |

Module packages must contain a top-level `icons` file.

## Project status

IconEditor is still under active development. Feedback and suggestions are welcome.
